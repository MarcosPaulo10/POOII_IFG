COMO FAZER OS TESTES E PRINTS - PASSO A PASSO
=============================================

Voce precisa de 2 coisas rodando: PostgreSQL (ja esta) e Java 17.


PARTE 1 - INSTALAR JAVA (so uma vez)
-------------------------------------
1. Abra o link: https://adoptium.net/temurin/releases/?version=17
2. Baixe o Windows x64 JDK (.msi)
3. Instale clicando Next, Next, Finish
4. Reinicie o computador


PARTE 2 - CONFIGURAR O BANCO (so uma vez)
------------------------------------------
1. Abra o pgAdmin (veio com o PostgreSQL)
2. Conecte no servidor local
3. Clique com botao direito em Databases > Create > Database
4. Nome: jdbc
5. Salve

Se a senha do postgres nao for "aluno", abra o arquivo:
   src/main/resources/application.properties
e troque a linha spring.datasource.password


PARTE 3 - SUBIR A API
----------------------
1. Va na pasta: Semanas\Semana_15\biblioteca
2. De duplo clique em: iniciar-api.bat
3. Espere aparecer "Started BibliotecaApplication"
4. NAO FECHE essa janela


PARTE 4 - TESTAR E TIRAR PRINTS (mais facil)
---------------------------------------------
Opcao A - Pelo navegador (so GET):
   Abra: http://localhost:8080/sistema-biblioteca/autores
   Tire print com Win+Shift+S

Opcao B - Pelo script automatico (recomendado):
   1. De duplo clique em: testar-api.bat
   2. Aperte Enter a cada teste
   3. Tire print da tela com Win+Shift+S
   4. Sao 11 prints no total

Opcao C - Pelo Postman:
   1. Baixe: https://www.postman.com/downloads/
   2. Abra o Postman > Import
   3. Selecione: Biblioteca-API.postman_collection.json
   4. Clique em cada requisicao e em Send
   5. Tire print mostrando URL, body e resposta


ORDEM DOS TESTES
----------------
1. POST autor      -> status 201
2. GET autores     -> status 200
3. GET autor/1     -> status 200
4. POST livro      -> status 201
5. GET livros      -> status 200
6. GET livro/1     -> status 200
7. GET livros/autor/1 -> status 200
8. PUT autor/1     -> status 200
9. PUT livro/1     -> status 200
10. DELETE livro/1 -> status 204
11. DELETE autor/1 -> status 204
